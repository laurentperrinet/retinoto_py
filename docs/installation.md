# Installation

## Stable release

To install Foveated Retinotopy, run this command in your terminal:

```sh
uv add retinoto_py
```

Or if you prefer to use `pip`:

```sh
pip install retinoto_py
```

## From source

The source files for Foveated Retinotopy can be downloaded from the [Github repo](https://github.com/laurentperrinet/retinoto_py).

You can either clone the public repository:

```sh
git clone git://github.com/laurentperrinet/retinoto_py
```

Or download the [tarball](https://github.com/laurentperrinet/retinoto_py/tarball/master):

```sh
curl -OJL https://github.com/laurentperrinet/retinoto_py/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd retinoto_py
uv pip install .
```
